How can we create forms in html webpage

In order to create form in html we use form tag The HTML | action Attribute is used to specify where the form data is to be sent to the server after the submission of the form. It can be used in the form element. There are two types of form action value :-
1-- Get Value
used to request data from a specified resource.
2-- Post Value
used to send data to a server to update a resource.

Grid is one of the way to make layouts in webpage. When we apply display property as flex the items arranges themseleves in horizontal direction. But when we apply display property as grid then the it will be shown in row column form. In grid property we can define the number of column and rows

To define the number of column we have to use grid-tempelate-column property.

The speciality about grid is that within the grid we can place any element anywhere

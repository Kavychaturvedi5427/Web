I am a box

In order to give Shadows we have to use box-shadow property the syntax of box Shadow is (box-shadow:h-offset,v-offset,blur,spread,color,inset).

  • h-offset and v-offset :: Determines the shadow's in horizontal and vertical position.
  • blur :: The larger the value the blurrier the shadow.
  • spread :: Expand or shrink the shadow size.
  • color :: Define the shadow color.
  • inset :: Makes the shadow inner.

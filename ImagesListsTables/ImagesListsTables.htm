HTML Lists

There are three types of lists in HTML

1-- unordered lists 2-- ordered lists 3-- defination lists

unordered lists (ul tag is used) creates list in an unordered manner with bullets on the list items
  • number one
  • number two
  • number three
  • number four


ordered lists (ol tag is used) creates list in an ordered manner with numbers/ roman number at the beginning of the list items
  1. number one
  2. number two
  3. number three
  4. number four


A Definition List in HTML is used to represent a list of terms along with their corresponding descriptions or definitions. The Definition List is created using the "dl" (Definition List) element, which wraps around one or more pairs of "dt" (Definition Term) and "dd" (Definition Description) elements.

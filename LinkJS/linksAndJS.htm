How to link css file

They are three types of css files we use :-

  1. Inline css -- Such type of css are used on the particular tag where we want to change appearance of the paragraph.
  2. Internal css -- Such type of css are declared under head tag using style tag.
  3. External css -- This type of Css is are externally linked using link:css tag under head tag.

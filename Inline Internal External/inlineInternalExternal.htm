How we can apply CSS in HTML webpage?

There are three ways to apply CSS in HTML webpage :-

  1. Inline CSS
  2. Internal CSS
  3. External CSS

How to apply inline CSS?

In order to apply Inline css you need to use style attribute under the tag where you want to apply CSS.

How to apply Internal CSS?

In order to apply internal CSS you need to use style tag under head tag and target using id/classes/tags.

How to apply External CSS?

In order to apply External CSS you need to create a css file and have to link it with the html file using Link:css attribute under head tag.
